Pengertian CSS

CSS (Cascading Style Sheet) merupakan bahasa yang digunakan untuk mengatur tampilan pada laman web . Pada materi sebelumnya kita mempelajari HTML, HTML merupakan building blocks of the web. Hampir semua eleman yang tampil pada web browser kita di buat menggunakan kode HTML. Mulai dari judul artikel, tombol, sampai form. Secara garis besar, jika HTML merupakan elemen pada tampilan web, maka CSS adalah style atau bagaimana elemen tersebut di tampilkan. Contohnya: Jika kita membuat elemen tombol menggunakan kode HTML, maka CSS dapat menentukan warna dan ukuran dari tombol tersebut.

Hal yang harus dilakukan:

1.Membuat file html

2.Mengetikkan isi html

3.Membuat file css

4.Mengetikkan isi style untuk file html

5.Menggabungkan file css dengan file html
